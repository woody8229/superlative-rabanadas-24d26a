PlateShift Netlify static site

Upload instructions (Netlify drag & drop):
1) Upload ALL files in this folder (netlify_site/) directly to Netlify.
2) Do NOT upload the folder itself nested inside another folder.

After uploading, update these placeholders:
- App Store URL and Google Play URL in index.html (#download section)
- Support email in support.html / delete-account.html / privacy.html (support@plateshift.app)

Branding assets:
- This site expects `plateshift_logo.png` to exist in this folder (copied from assets/images/plateshift_logo.png).

Notes:
- This site is standalone (not the Flutter web build).
- It’s intended for marketing + legal URLs required by Apple/Google (Privacy Policy, Terms, Support, Account deletion).
